# User Authentication System

An ASP.NET Core MVC application demonstrating secure user authentication and authorization using ASP.NET Core Identity. The system includes user registration, login, logout, and protected pages backed by a SQL database.

## Features
- User registration and login
- Secure password hashing (ASP.NET Identity)
- Authentication and authorization
- Protected pages accessible only to authenticated users
- SQL database with Identity tables
- Clean MVC architecture

## Technologies Used
- ASP.NET Core MVC (.NET 8)
- C#
- ASP.NET Core Identity
- Entity Framework Core
- SQL Server (LocalDB)
- Razor Views
- Bootstrap
- Visual Studio
- Git & GitHub

## Project Purpose
This project was built to demonstrate authentication and authorization in ASP.NET Core applications using industry-standard security practices. It complements other MVC projects by focusing on user access control and secure routing.

## Key Concepts Demonstrated
- Authentication vs Authorization
- ASP.NET Identity integration
- Protected routes using `[Authorize]`
- Secure user management
- MVC architecture
